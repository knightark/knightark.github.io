C#使用Jenkins持续集成学习笔记

在win10中安装使用Jenkins

##一 安装Git

首先安装Git-2.17.0-64-bit.exe，安装的过程傻瓜式安装，一路点击是。

注意：安装Git时的安装目录需要一个纯英文的，没有特殊符号的目录

我的安装目录是C:\software\Git

##二 安装MSbuild

安装VS的时候就安装了MSBuild，如果没有MSBuild，可以到微软官网下载一个安装。

我的MSBuild安装目录是C:\Program Files (x86)\MSBuild\14.0\Bin

##三 安装Inno Setup

安装innosetup-6.0.3.exe，一路下一步。
